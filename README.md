# VQGAN_Japanese-CLOOB

## About this repository
rinnaが公開した日本語版CLIPとVQGANを用いた日本語からの画像自動生成を行う。

Automatic generation of images from Japanese with CLIP in Japanese published by rinna and VQGAN.

## Preparation
```bash
bash init.sh
```
